About Project

AlertBox is a Machine Learning-based web application that detects whether an SMS message is Spam or Not Spam.

The model analyzes text content (and optionally, text extracted from uploaded images) to identify suspicious patterns and alert users before they fall for phishing or scam messages.

Built using Python (Flask), Scikit-learn, and Natural Language Processing (NLP) techniques, AlertBox aims to enhance digital communication security by providing an accurate and user-friendly spam detection experience.
